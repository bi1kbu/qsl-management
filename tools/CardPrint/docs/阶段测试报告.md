## 实机配合事项（仅打印相关）

以下操作需要你在有针式打印机的 Windows 主机上配合一次：

1. 安装打印机驱动并确认系统“设备和打印机”可见目标打印机。  
2. 运行 `python -m cardprint.cli printer list`，确认真实打印机名称。  
3. 运行标定 UI：`python -m cardprint.cli ui calibrator`，点击“打印标定页”，测量十字偏移并填入。  
4. 保存预设后运行打印 UI：`python -m cardprint.cli ui printer`，加载预设并打印 1 条样例确认落点。  
5. 最后导入 10-100 行 CSV/XLSX 做批量验证，核对逐行结果与实体输出。
