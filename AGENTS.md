# 项目目标与范围
- 本仓库为基于 Halo v2 插件框架下的QSL卡片管理系统（QSL是业余无线电的通联确认卡片）。

# 开发过程中的注意事项
- 所有技术方案和框架需要符合 Halo v2 的技术要求（访问 https://docs.halo.run/developer-guide/plugin/basics/structure 等页面进行查阅）
- 在开始之前你需要先了解一下halo的基本知识（https://docs.halo.run/developer-guide/core/framework 等页面）
- 开发过程当中使用 development 模式进行（https://docs.halo.run/developer-guide/plugin/basics/manifest）
- 当前主机已配置docker desktop，请优先使用docker运行开发环境（https://docs.halo.run/developer-guide/plugin/prepare）
- 先创建插件项目的框架，再开始后续开发（https://docs.halo.run/developer-guide/plugin/hello-world）
- 项目需求文件请见 `需求描述.md`
